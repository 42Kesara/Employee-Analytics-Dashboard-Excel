# 📌 Project Overview

Organizations generate large volumes of employee data, making it difficult to quickly identify workforce trends and performance metrics.

This dashboard provides an interactive HR analytics solution that enables users to monitor employee statistics, salary information, department performance, employee ratings, and geographical distribution using Microsoft Excel.

The dashboard is fully interactive through slicers, allowing decision-makers to analyze data dynamically without writing formulas.

# 🖼 Additional Features

## 🔍 Employee Search Panel

<p align="center">
  <img src="search-panel.png" width="1000">
</p>

The Employee Search Panel allows users to quickly retrieve employee details by entering or selecting an Employee ID.

This improves usability by providing fast access to key employee information such as name, department, city, salary, and rating.

---

## ⚡ Power Query

<p align="center">
  <img src="power-query.png" width="1000">
</p>

Power Query was used to prepare and transform the dataset before analysis.

Key transformations include:

- Importing employee data
- Cleaning inconsistent records
- Removing duplicates
- Formatting columns
- Preparing data for reporting
- Creating a refreshable workflow

---

## 📊 Power Pivot Data Model

<p align="center">
  <img src="power-pivot.png" width="1000">
</p>

Power Pivot was used to create relationships between multiple tables and build a scalable data model.

Example relationships:

- Employees → Departments
- Employees → Salary Grades

This structure supports efficient reporting and future DAX-based analysis.
